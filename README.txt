# Snake
Jogo educacional trabalho avaliativo do final da disciplina de "Fundamentos da Programação" do primeiro semestre

Academico: Gabriel Vanz da Silva
E-mail: gabriel.vanz.silva@gmail.com

	Jogo feito com o propósito de incentivar as crianças a brincar com o game clássico Snake porém coletando letras do alfabeto
temos no jogo o obstáculo de uma bomba que vai surgindo aleatóriamente junto com a letra, cabendo a criança diferenciar a letra
da bomba. Conforme as letras vão sendo obtidas, a cobrinha vai crescendo e vai somando na pontuação.
